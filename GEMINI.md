# การเพิ่มปุ่มคัดลอกไฟล์ใน File Tools

## สิ่งที่ดำเนินการ
1. เพิ่มปุ่ม **Copy upscale img** พร้อมคำอธิบายใต้ปุ่มว่าทำหน้าที่คัดลอกไฟล์จากโฟลเดอร์ `upscayl_png_remacri-4x_4x` ไปยัง `4000x4000` (อ้างอิงและทำงานใน Path 1)
2. เพิ่มปุ่ม **Local to Remote** พร้อมคำอธิบายใต้ปุ่มว่าทำหน้าที่คัดลอกโฟลเดอร์ `4000x4000` และ `Sticker Set` จาก Path 1 ไปยัง Path 2
3. เพิ่มคำอธิบายการทำงานใต้ปุ่ม `Create Folders` และ `Unzip Downloads` ในหมวด File Tools ด้วย เพื่อให้รูปแบบการแสดงผลของปุ่มทั้งหมดเหมือนกันและเข้าใจง่าย
4. สร้างฟังก์ชันการจัดการไฟล์ `action_copy_upscale` และ `action_local_remote` ซึ่งใช้ไลบรารี `shutil.copy2` ในการคัดลอกไฟล์ได้อย่างถูกต้อง
